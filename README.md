# 🧠 CIFAR-10 Image Classifier — CNN from Scratch

A Convolutional Neural Network built completely from scratch
using PyTorch to classify images into 10 categories.

**Achieved ~85% test accuracy** on 10,000 unseen images.

---

## 📊 Results

| Metric | Value |
|--------|-------|
| Test Accuracy | 85.2% |
| Training Epochs | 30 |
| Optimizer | Adam (lr=0.001) |
| Total Parameters | ~1.2M |
